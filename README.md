# JSFiddle to Git
Convert a JSFiddle to Git repository.

## Installation

```sh
$ git clone https://github.com/IonicaBizau/jsfiddle-to-git
$ cd jsfiddle-to-git
$ ./run.sh
```

In the future it should be possible using:

```sh
$ sudo npm install -g jsfiddle-to-git
```

## How to use

```sh
$ jsfiddle-to-git
```
